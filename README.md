# 🌸 AI Flower Test Bot

Telegram бот для определения психологического типа личности через визуальные предпочтения. 
На основе ответов генерирует AI-промпт для создания персонализированного букета.
---
## _Легенда бота_
Бот маскируется под "Лабораторию психологии личности" - пользователи думают, 
что проходят тест на определение характера, хотя на самом деле мы собираем данные 
для создания идеального букета-сюрприза.

## 🚀 Быстрый старт

### Локальный запуск
```bash
# Клонируем репозиторий
git clone https://github.com/yourusername/flower-test-bot.git
cd flower-test-bot

# Устанавливаем зависимости
go mod download

# Создаем .env файл
cp .env.example .env
# Редактируем .env - добавляем TELEGRAM_BOT_TOKEN

# Запускаем
go run main.go
